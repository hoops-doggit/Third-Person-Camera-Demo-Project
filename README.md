# Third Person Camera System - Demo Project
An easy to use and extendable third person camera system built for Unity.

In active development but ready to use as a solid base for your own work.

Built to accompany my talk on how to build your own camera system:
https://youtu.be/NoRu4ix8b94

Includes:
* Camera Manager
* Simple Orbit camera
* Player Camera
* Angle Spring
* Float Spring
* Collison handling
* Continuous system debugger
* Basic character controller and game world to demonstrate features.
